# Library Management System

A desktop application developed in **C#** for managing library operations.  
The system enables librarians to maintain records of library members and books, as well as track borrowing and returning activities.


## Features

The system supports the following functionalities:

- Create new library members
- Edit member information
- Search members by personal data
- Delete members from the system
- Add new books to the library catalog
- Remove books from the system
- Search books by criteria
- Borrow books to library members
- Return borrowed books


## Actors

### Librarian
The main user of the system responsible for managing members, books, and borrowing records.

### Library Members
Members can use the system within the library to search for available books.


## Use Cases

The system includes the following main use cases:

1. Create new member  
2. Update member information  
3. Search members  
4. Delete member  
5. Add book  
6. Remove book  
7. Search books  
8. Borrow book  
9. Return book  


## Project Goal

The project focuses on applying **software design concepts**, including:
- use case modeling
- system functionality design
- structured implementation in C#
